# 🌐 Portfolio

A modern and interactive Personal Portfolio Website built using HTML5, CSS3, and JavaScript. The project showcases my skills, education, coding profiles, and projects with a clean user interface .

## 🚀 Live Demo

🔗 Website: https://portfolio-orcin-eight-g71fxqjh3r.vercel.app

## 📂 GitHub Repository

🔗 https://github.com/Sahill1122/Portfolio

## ✨ Features

* Interactive kinetic grid background animation
* Responsive and modern user interface
* About Me section with educational details
* Skills and technical expertise showcase
* Coding profiles integration (LeetCode, GitHub, LinkedIn)
* Smooth animations and hover effects
* Clean and minimal portfolio design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Google Fonts
* Git & GitHub
* Vercel

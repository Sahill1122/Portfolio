# 🌐 Portfolio

A modern, responsive Portfolio Website built using HTML5, CSS3, and JavaScript. The website showcases my skills, projects, education, and contact information with a clean user interface and smooth user experience across desktop and mobile devices.

## ✨ Features

* Responsive design for all screen sizes
* Modern and clean user interface
* Smooth scrolling navigation
* About Me section
* Skills showcase section
* Projects section with project cards
* Contact section with social links
* Interactive animations and hover effects
* Mobile-friendly design

* CSS3
* JavaScript (ES6)
* Font Awesome
* Google Fonts
